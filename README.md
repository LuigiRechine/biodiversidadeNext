# 🌳 Projeto Biodiversidade - Cadastro de Árvores

Bem-vindo ao Projeto Biodiversidade, uma aplicação web para cadastro e visualização de espécies de árvores 🌱  
O objetivo é facilitar o registro e a organização de informações sobre a flora local, contribuindo para a preservação da biodiversidade.

🔗 Acesso: https://ibirabio.vercel.app/

---

## 🔹 Funcionalidades

- Visualização de espécies em cards com imagem  
- Nome comum e nome científico  
- Tipo e porte da árvore  
- Cadastro de novas espécies  
- Busca em tempo real  
- Interface responsiva (desktop e mobile)

---

## 🖥️ Tecnologias Utilizadas

- Next.js (React Fullstack)  
- Supabase (PostgreSQL + API)  
- CSS3  
- Vercel (Deploy)

---

## 📌 Estrutura do Projeto

/app → páginas e rotas  
/app/api → backend integrado  
/lib → conexão com Supabase  
/components → componentes reutilizáveis  
/public → arquivos estáticos
